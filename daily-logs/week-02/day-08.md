## 📖 Key Concepts

### Recursion Basics
- Function calling itself
- Base case: Condition to stop
- Recursive case: Function calls itself
- Call stack: How recursion works
- Stack overflow: Too deep recursion

### Backtracking
- Explore all possibilities
- Undo choices that don't work
- Use recursion + pruning
- Applications: N-Queens, Sudoku, Permutations
