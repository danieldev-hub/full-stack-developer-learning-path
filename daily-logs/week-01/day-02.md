## 📖 Key Concepts

### Stacks (LIFO - Last In First Out)
- Push: Add to top - O(1)
- Pop: Remove from top - O(1)
- Peek: View top - O(1)
- Applications: Browser back button, undo/redo, bracket matching

### Queues (FIFO - First In First Out)
- Enqueue: Add to back - O(1)
- Dequeue: Remove from front - O(1)
- Peek: View front - O(1)
- Applications: Job scheduling, BFS, printer queue
