## 📖 Key Concepts

### Simple Sorts (O(n²))
- **Bubble Sort**: Compare adjacent, swap if needed
- **Selection Sort**: Find min, place at beginning
- **Insertion Sort**: Build sorted array one by one

### Efficient Sorts
- **Merge Sort**: O(n log n), Divide & Conquer, Stable
- **Quick Sort**: O(n log n) avg, O(n²) worst, In-place
- **Heap Sort**: O(n log n), In-place
